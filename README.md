# Data Analytics Project

**Overview**
This project demonstrates an end-to-end data analytics workflow, covering data ingestion, exploratory data analysis (EDA), data cleaning, SQL-based analysis, dashboard creation, and final reporting. The goal is to extract meaningful insights from raw data and present them in a clear, actionable format for decision-making.

**Dataset**
* The dataset used in this project contains structured data relevant to the analysis domain (e.g., sales, customers, operations).
* It includes multiple variables such as numerical and categorical.
* Data quality issues such as missing values, duplicates, and inconsistencies were identified and addressed during preprocessing.
  
**Tools & Technologies**

* **Python**: Data loading, cleaning, and EDA (Pandas)
* **MS SQL Server**: Querying and data analysis
* **Power BI**: Interactive dashboard creation
* **Gamma**: Presentation (PPT) creation
* **Jupyter Notebook**: Development and documentation
  
## Project Steps

 **1. Data Loading**

* Imported dataset using Python (Pandas)
* Inspected structure, data types, and initial summary statistics

 **2. Exploratory Data Analysis (EDA)**

* Analyzed distributions, trends, and relationships
* Visualized key patterns using charts and graphs
* Identified anomalies and outliers

**3. Data Cleaning**
* Handled missing values
* Removed duplicates
* Standardized formats and corrected inconsistencies
* Created new derived features where necessary

### 4. SQL Analysis

* Loaded cleaned data in SQL Server
* Wrote SQL queries to:

  * Aggregate data
  * Filter and segment datasets
  * Perform joins across tables
  * Generate key business metrics

**5. Dashboard Creation**

* Built an interactive Power BI dashboard
* Included KPIs, filters, and drill-down capabilities
* Focused on clear and intuitive data storytelling

**6. Reporting & Presentation**

* Created a structured analytical report summarizing findings
* Designed a presentation (PPT) using Gamma
* Highlighted insights, trends, and recommendations
  
 **Dashboard**
* Interactive visualizations for key metrics
* User-friendly filters and slicers
* Clear representation of trends and comparisons
* Designed for both technical and non-technical stakeholders

**Results & Insights**
* Identified key trends and patterns in the dataset
* Highlighted areas of improvement and opportunities
* Provided actionable recommendations based on data analysis
* Improved data understanding through visualization and reporting

**How to Run**
### Prerequisites
* Python (3.x)
* Jupyter Notebook
* MS SQL Database
* Power BI Desktop

**Steps**
1. Clone the repository
2. Install required Python libraries:
   pip install pandas 
3. Open the Jupyter Notebook and run all cells for EDA and cleaning
4. Load cleaned data into your SQL database
5. Execute SQL scripts provided in the `/sql` folder
6. Open the Power BI file (`.pbix`) to explore the dashboard
7. Review the report and presentation files for insights

**Conclusion**
This project showcases a complete data analytics pipeline, from raw data processing to business insight generation. It highlights proficiency in Python, SQL, data visualization, and storytelling—key skills for data-driven decision-making.
